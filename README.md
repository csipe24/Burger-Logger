# Burger-Logger Application

## Description
Welcome to the Burger Logger application! This app is deployed via heroku and uses a database of different burger types! Each burger datapoint consists a name, id and boolean value of devoured.
The application uses object relational mapping to create get, post and update requests from the burger database.

# Usage
Use this application for a quick and easy way enter and log which burgers you have eaten! This application uses several packages including express, express handlebars and mysql!
